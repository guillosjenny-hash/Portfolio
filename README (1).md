# Jenny Rose T. Guillos — Personal Portfolio Website

A personal portfolio website built to showcase my profile, skills, projects, and contact information as part of my IT coursework.

## Description

This is a fully responsive, single-page portfolio website designed with a modern dark theme. It features smooth scroll animations, a fixed navigation bar, and sections covering my background, technical and soft skills, academic projects, education, and a contact form.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables, Animations)
- JavaScript (Vanilla)
- [Font Awesome 6.5](https://fontawesome.com/) — icons
- [Google Fonts](https://fonts.google.com/) — Syne & DM Sans
